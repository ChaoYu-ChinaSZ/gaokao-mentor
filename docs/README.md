# 高考志愿 Mentor 系统 — 完整说明

## 项目背景

高三学生在填报志愿时面临一个根本矛盾：对自己的认识不足，却要做出影响未来数年甚至一生的选择。市场上的工具大多数是"输入分数，输出志愿表"，但没有人帮助学生先回答一个更根本的问题——**"我到底是谁，我喜欢什么，我能做什么"**。

高考志愿 Mentor 的目标，就是填补这个空白。

---

## 核心理念

### 传统方式 vs 我们的方式

| 传统方式 | 高考志愿 Mentor |
|---|---|
| 告诉学生"你应该选什么" | 引导学生"你自己想选什么" |
| 输入分数，输出院校列表 | 先认识自己，再匹配合适的方向 |
| 一次性结果 | 持续演进的自我认知过程 |
| 焦虑驱动（"不选好就完了"）| 好奇驱动（"我想更了解自己"）|

### 三个维度

系统的评估框架建立在三个维度上：

1. **自我认知**（Ikigai + RIASEC）
   - 热爱什么
   - 擅长什么
   - 职业兴趣分布

2. **现实约束**（家庭探测）
   - 家庭期待与反对
   - 经济承受能力
   - 家庭结构（是否独生子女）

3. **人生节奏**（Schein 职业锚推断）
   - 工作与生活的平衡偏好
   - 长期稳定 vs 短期拼搏
   - 工作是一辈子还是阶段

## 对话流程详解

### 第一阶段：Ikigai 四问（约 10 分钟）

通过四个问题，帮助学生梳理：

1. **热爱** — 做什么事情的时候，你感觉时间过得很快，完全忘记了别的事？
2. **擅长** — 什么事情你稍微一认真，就比大多数人做得好？
3. **使命** — 你觉得这个世界最需要什么？哪怕没有钱，你也愿意做的？
4. **愿景** — 想象 5 年后的理想生活：你在哪里，和谁在一起，在做什么？

### 第二阶段：现实约束探测（约 5 分钟）

六个问题，探测真实的外部约束：

- 家里人对你学什么专业，有没有明确期待或反对？
- 经济角度看，有哪些是承担不起的？
- 是不是独生子女？
- 工资高没时间 vs 工资一般但准时下班，你选哪个？
- 拼命工作几年然后自由，还是一直平稳？
- 工作是一辈子的事，还是一个阶段的事？

### 第三阶段：RIASEC 职业兴趣测评（约 15 分钟）

25 道二选一的情境题，量化六个职业兴趣维度：

- **R** 实际型 — 喜欢动手做事
- **I** 研究型 — 喜欢追根究底
- **A** 艺术型 — 喜欢创意表达
- **S** 社会型 — 喜欢帮助别人
- **E** 企业型 — 喜欢影响说服
- **C** 常规型 — 喜欢规整有序

### 第四阶段：清晰度判断 + 推荐

根据 RIASEC 分布的离散程度，判断清晰度：

| 清晰度 | 表现 | 建议 |
|---|---|---|
| **清晰** | 一两个维度特别突出 | 直接锚定方向，深入探索 |
| **初步清晰** | 两三个维度都不错 | 关注维度组合，窄口切入 |
| **模糊** | 分布较均衡 | 广泛探索，不急于定方向 |

## 推荐专业的方法

推荐不是查表，而是根据 RIASEC 维度组合，结合职业锚推断，生成个性化的专业方向建议。

## 与市场上产品的差异

| 产品 | 方法 | 我们的差异 |
|---|---|---|
| 志愿填报APP | 分数+历年数据→推荐院校 | 我们先做自我认知 |
| 张雪峰式咨询 | 专家经验+实用主义 | 我们不给结论，给工具 |
| MBTI测试 | 16型人格标签 | 我们不贴标签，展示维度 |
| 霍兰德测评 | RIASEC原型匹配 | 我们加了现实约束层 |

## 适用场景

- 高一/高二学生：提早规划，有充足时间探索
- 高三学生：出分前的自我梳理
- 志愿填报机构：作为咨询前的预热工具
- 学校心理老师：班级团体辅导的辅助工具

---

## 局限性与诚实边界

1. **不提供院校录取预测** — 本系统不做分数-院校匹配（V2.0 计划支持）
2. **不替代专业咨询** — 对于有复杂家庭情况的学生，系统仅作为辅助参考
3. **依赖于学生诚实回答** — 答案质量决定推荐质量
4. **职业锚是内部推断** — 不让学生做完整量表，而是从对话中推断
