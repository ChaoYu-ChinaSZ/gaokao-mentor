## 快速开始

### 场景一：学生直接使用

如果你已经在微信上添加了机器人：

```
1. 打开微信
2. 找到 [机器人名称]
3. 发送：高考咨询

```

全程约30-40分钟，可以分多次完成。

## 学生使用指南

### 这是什么

一个AI对话工具。它不问你的成绩、不查历年分数线，
而是通过四个问题帮助你理清：**我喜欢什么、擅长什么、想过什么样的生活**。

根据你的回答，生成推荐的专业方向和职业兴趣报告。

### 整个流程是什么样的

```
第一步：4个问题（Ikigai）
   ↓ 约10分钟
第二步：6个问题（现实约束）
   ↓ 约5分钟
第三步：25道选择题（RIASEC职业兴趣）
   ↓ 约15分钟
第四步：获得推荐结果
   ↓
可选：填写反馈（30秒）
```

### 第一步：Ikigai 四问

你会收到4个问题，每个都需要认真思考后回答：

**Q1：什么时候时间过得很快？**
> 做什么事情的时候，你感觉时间过得很快，完全忘记了别的事？
> （比如：打游戏、和朋友聊天、解数学题、写代码、画画……）

**Q2：什么事情你稍微认真就比人强？**
> 什么事情你稍微一认真，就比大多数人做得好？
> （不需要刻意练习，只要认真就能超过平均的那种）

**Q3：这个世界最需要什么？**
> 你觉得这个世界最需要什么？哪怕没有钱，你也愿意做的？
> （可以是很大的事，也可以很小的事）

**Q4：5年后理想生活**
> 想象5年后的理想生活：你在哪里，和谁在一起，在做什么？

认真回答，不要想"标准答案"，没有对错。

---

### 第二步：现实约束

6个简短问题，了解你的真实情况：

- 家里人对你学什么专业有没有明确期待或反对？
- 经济角度有没有承担不起的选项？
- 是不是独生子女？
- 工资高没时间 vs 工资一般准时下班，你选哪个？
- 拼命工作几年然后自由，还是一直平稳？
- 工作是一辈子的事，还是一个阶段的事？

如实回答。这些信息会影响最终推荐的个性化程度。

---

### 第三步：RIASEC 职业兴趣测评

25道二选一的情境题。

每道题给你描述两个场景，选你觉得更想做的那个。

没有对错，喜欢就是喜欢，不喜欢就是不喜欢。

---

### 第四步：获得推荐

完成后，你会收到：

**1. RIASEC 雷达图**
六个维度的分布图，展示你的职业兴趣画像

**2. 推荐专业方向**（3-5个）
根据你的回答生成的方向建议

**3. 个性化备注**
基于你的现实约束给出的参考说明

---

### 注意事项

- **可以中途停止**：直接关闭对话，下次发"继续咨询"从上次位置继续
- **可以重新开始**：发"重新开始"会清空档案重新跑
- **结果仅供参考**：这是帮你思考的工具，不是权威结论
- **隐私安全**：你的回答只存在本地，不上传

## 机构运营指南

### 适合场景

- 高中班主任/辅导员，想给学生做自我认知辅导
- 志愿填报机构，想作为预热工具
- 学生社团，想做志愿填报分享活动
- 教育博主，想提供互动内容给粉丝

### 使用方式

**方式A：单人体验（最简单）**

把你的微信号分享给学生，让学生关注后自行体验。

适合：分散的个体用户

**方式B：集体引导（适合班级/社群）**

1. 建一个微信群，把学生拉进来
2. 在群里发使用说明
3. 学生逐一体验（每人30-40分钟）
4. 收集学生的推荐结果（自愿分享）
5. 汇总反馈，形成班级群体画像

**方式C：嵌入活动（适合分享会/讲座）**

在志愿填报分享活动中，用这个工具做现场演示：

1. 找一个志愿者现场体验完整流程
2. 投影展示对话过程
3. 结束后邀请其他学生提问和体验

---

### 如何收集学生反馈

1. **轻量反馈**：对话结束后系统自动弹出，30秒完成
2. **深度反馈**：对话结束后选填，2分钟完成
3. **匿名反馈**：扫码填问卷（问卷链接见GitHub仓库）

---

### 数据使用说明

- 学生数据完全匿名，机构无法获取个体回答内容
- 机构只能看到汇总后的群体画像（如"本班63%学生推荐I研究型"）
- 如需个体完整档案，由学生自行保管

## 技术部署指南

### 完整部署步骤

#### 前置条件

- Linux/macOS/WSL 环境
- Python 3.12+
- Git
- Hermes Agent（参考 [hermes-agent](https://github.com/ChaoYu-ChinaSZ/hermes-agent)）
- 微信/飞书/钉钉 等消息平台账号

#### Step 1：安装 skill

```bash
git clone https://github.com/ChaoYu-ChinaSZ/gaokao-mentor.git ~/.hermes/skills/gaokao-mentor
```

#### Step 2：测试引擎

```bash
python3 ~/.hermes/skills/gaokao-mentor/scripts/test_engine.py
```

看到以下输出即成功：

```
测试通过！
模拟推荐专业：['计算机科学与技术', '电子信息']
推荐职业锚：AU, CH, TF
```

#### Step 3：确保 Hermes Agent 运行中

```bash
hermes agent status
hermes gateway status
```

#### Step 4：确认微信已接入

gateway 状态应显示 `weixin: connected`

#### Step 5：重启 agent 使 skill 生效

```bash
hermes agent restart
```

#### Step 6：验证 skill 可用

在微信中向机器人发送：

```
高考咨询
```

应收到 Ikigai 第一问。

---

### 对接其他消息平台

如果你不使用微信，而是使用飞书/钉钉/Telegram：

1. 参考 [hermes-agent 文档](https://github.com/ChaoYu-ChinaSZ/hermes-agent) 配置对应平台 adapter
2. 平台接收消息后，统一通过 Hermes Agent 处理
3. gaokao-mentor skill 的触发逻辑不依赖于具体平台

## 常见问题

### Q: 学生在对话中途停止了，会丢失数据吗？

不会。每次回答后数据自动保存。下次发"继续咨询"，从上次位置继续。

---

### Q: 推荐结果准吗？

没有工具的推荐是100%准确的。这个系统的设计目标是**帮助你更了解自己**，而不是给你一个标准答案。推荐结果应该和你的自我认知对照参考，而不是盲目服从。

---

### Q: 成绩不好，这个工具有用吗？

这个工具完全不看成绩。它关注的是：**你的兴趣、性格、价值观和生活偏好**。即使成绩暂时不理想，了解自己适合什么方向，对志愿填报和未来规划同样有价值。

---

### Q: 家里人意见很大，这个工具怎么应对？

工具不会替你做决定，但会帮助你**把自己的真实想法表达清楚**。在现实约束探测阶段，你可以把家庭因素完整表达，最终推荐会附带这些约束备注。你可以用工具的结论和家里人更理性地沟通。

---

### Q: 一个人能用多次吗？

可以。每次跑完流程后，档案会更新。如果过段时间有了新的想法，发"重新开始"可以清空档案重新跑。多次使用可以看到自己认知的变化。

---

### Q: 想提建议/反馈，怎么操作？

三种方式：
1. 在微信里直接发"反馈"，告诉我你的想法
2. 在 GitHub Discussions 中发帖：https://github.com/ChaoYu-ChinaSZ/gaokao-mentor/discussions
3. 扫描 GitHub README 底部的匿名反馈二维码

---

### Q: 技术报错/无法运行怎么办？

1. 先确认 `test_engine.py` 是否能正常运行
2. 查看 GitHub Issues 是否已有相同问题
3. 如无，创建新 Issue，附上报错信息和你的环境信息
